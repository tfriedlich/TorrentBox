# TorrentBox
These are my notes for creating my rPi based torrent box. This should include:
* Radarr
* Sonarr
* Deluge

1. Create new user
   - sudo adduser [todd]
   - sudo adduser [todd] sudo
  
2. Update and Upgrade
   - sudo apt-get update
   - sudo apt-get upgrade
  
3. Install Sonarr
   - Update Mono
     - sudo apt update && apt install curl mediainfo
